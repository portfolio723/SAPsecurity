# **App Name**: SAP Security Expert Hub

## Core Features:

- Homepage Navigation: Sticky navigation bar for easy access to key sections: content, login, subscription.
- Content Cards: Display SAP security articles using grid-based cards with category labels, titles, micro-descriptions, and read times.
- Newsletter Subscription: Implement a clean, non-intrusive newsletter CTA to capture secondary email addresses.

## Style Guidelines:

- Primary color: SAP Blue (#003472) to maintain brand recognition and trust.
- Background color: Light blue (#EBF0F5), a desaturated version of the primary hue to ensure reading comfort.
- Accent color: Sharp Red (#F25140) for CTAs. To create a bold visual impact, the contrast with the light background is carefully planned, drawing the user's attention effectively. This emphasis will significantly highlight conversion actions.
- Headings: 'Source Sans Pro', sans-serif.
- Body: 'Inter', sans-serif.
- Maintain a content-focused design. All the articles and CTAs will be sharp edged with no radius
- Implement mobile-responsive design using stacked layouts and thumb-friendly CTAs, with breakpoints at desktop (≥ 1280px), tablet (≥ 768px), and mobile (≤ 767px).
